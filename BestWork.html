Resume History of Web Best Work



Jedd's Tree Care

Jedd's Tree care was an assignment given to us by our teacher. It helped me learn a lot about cool customization features that I had not seen before. I really enjoy the way this assignment turned out.



Curbside Thai

Curbside Thai was another great assignment that progressed my knowledge on the crazy amount of different tidbits you need to know when creating a website. Curbside Thai introduced me to using Hypertext links which came in handy when it came time for my final project.



Custom Treasure Hunt

This personally was my favorite assignment. The teacher had assigned us to create our own treasure map hunt using codes that he had shown us earlier. I created a baseball version of it using baseball terms that correlate to whether you're close, or too far away.
